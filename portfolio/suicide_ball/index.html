Suicide Ball

Juiciest Pong Clone Ever Invented

Goals

I wanted to brush up on C++ after a long absence, and I wanted to learn SFML 2, so I initially built a pong clone. Not satisfied, I decided I wanted to use HLSL to make it this pong clone gorgeous. I also wanted it to be visually impressive, so I came up with whole new concept that I hope makes the pong formula fun and entertaining, even if it's just for a while.
